## 📖 项目简介

这是一个使用 Vue 3 + Vite 构建的《光遇（Sky: Children of the Light）》游戏介绍网站。通过精美的页面设计和流畅的交互体验，向玩家展示光遇的游戏特色、世界观和社区文化。

## 🎮 关于光遇

《光遇》是由 thatgamecompany 开发的一款社交冒险游戏。玩家将化身为光之子女，在云端王国中探索七个梦幻国度，拯救堕落的星灵，点亮希望之光。游戏以其温馨治愈的美术风格、动人的音乐和独特的社交玩法，赢得了全球超过 1 亿玩家的喜爱。

### 游戏特色

- 🌟 **治愈温馨** - 温暖的视觉风格和动人的音乐，带来心灵的慰藉
- 👥 **社交互动** - 与来自世界各地的玩家相遇，建立珍贵的友谊
- 🗺️ **探索冒险** - 在七个梦幻国度中自由飞翔，发现隐藏的秘密
- 🎨 **自由定制** - 收集各种斗篷、面具和发型，打造独特的角色

### 七大国度

1. **晨岛** - 新手的起点，学习飞翔的地方
2. **云野** - 广阔的草原和浮岛，充满自由的气息
3. **雨林** - 茂密的森林和神秘的树屋
4. **霞谷** - 竞速赛道和冰雪滑道的欢乐天地
5. **墓土** - 失落的战场，充满挑战
6. **禁阁** - 知识的圣殿，保存着古老的记忆
7. **暴风眼** - 最终的试炼之地

## 🚀 快速开始

### 环境要求

- Node.js 16.x 或更高版本
- npm 或 yarn 或 pnpm

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

启动后访问 `http://localhost:5173` 即可查看网站。

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 🛠️ 技术栈

- **框架**: Vue 3 - 渐进式 JavaScript 框架
- **构建工具**: Vite - 下一代前端构建工具
- **语言**: JavaScript (ES6+)
- **样式**: CSS3 (含渐变、动画等现代特性)

## 📁 项目结构

```
vite-project/
├── public/             # 静态资源
│   └── vite.svg
├── src/
```

## 🎨 设计特色

- **梦幻配色** - 采用紫色渐变和温暖色调，呼应游戏的天空主题
- **流畅动画** - 悬浮、渐变等动效提升用户体验
- **响应式设计** - 完美适配桌面端和移动端
- **模块化布局** - 清晰的区块划分，便于浏览和维护
